# AgentOpsIQ Product Brief

## Product

**AgentOpsIQ Control Plane**

## Short Name

**AgentOpsIQ**

## Tagline

Production intelligence for reliable AI agents.

## Category

OpenTelemetry-native reliability, governance, and production-readiness platform for enterprise AI agents running on Azure/Kubernetes.

## Core Thesis

Enterprise AI agents need more than demos. They need production evidence: traces, logs, metrics, tool-call auditability, Kubernetes runtime context, deployment metadata, incident timelines, SLOs, policies, and governance controls.

AgentOpsIQ helps platform, SRE, DevOps, and AI infrastructure teams safely deploy, observe, debug, and govern AI agents in customer-owned enterprise environments.

## Initial Wedge

Start with the **AgentOpsIQ Scanner**:

- Scan AI agent Kubernetes deployments.
- Score production readiness.
- Detect missing OpenTelemetry, Kubernetes, CI/CD, security, cost, and governance controls.
- Generate a Markdown/JSON report.
- Run locally and in GitHub Actions.
