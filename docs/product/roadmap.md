# AgentOpsIQ Roadmap

## 0-7 Days

- Confirm legal and visa operating boundaries before commercialization.
- Keep development original and independent.
- Finalize repo structure.
- Define the first 20 readiness checks.
- Build local Docker Desktop and Ollama lab plan.

## 30 Days

- Build `agentopsiq scan`.
- Scan Kubernetes YAML and Helm values.
- Generate Markdown and JSON reports.
- Create sample AI agent and failure fixtures.
- Capture first OpenTelemetry traces locally.
- Refactor scanner checks into the shared rule engine without changing output.

## 60 Days

- Add GitHub Action wrapper.
- Add PR comment output.
- Add CI fail-on-critical mode.
- Add OpenTelemetry collector pack.
- Add local dashboard prototype.
- Prototype `agentopsiq guard` with cost-cap and block-risky-tool rules.

## 90 Days

- Add incident workbench MVP.
- Generate incident summaries with Ollama.
- Build AKS deployment outline.
- Create demo screenshots and a short product video.
- Collect feedback from trusted SRE/platform contacts.
- Add first web approval inbox requirements and audit-log design.

## 3-6 Months

- Package Helm chart and Terraform AKS reference.
- Create design partner assessment.
- Build policy engine prototype.
- Decide open-source boundary for scanner.

## 6-12 Months

- Multi-cluster control plane prototype.
- Agent registry.
- Reliability scorecards.
- RBAC, audit logs, and enterprise reporting.
- BYOC deployment path.

## 12-24 Months

- Enterprise integrations.
- Compliance policy packs.
- Open-source community wedge.
- Paid enterprise pilots.
- Commercial company formation if legally and commercially viable.
