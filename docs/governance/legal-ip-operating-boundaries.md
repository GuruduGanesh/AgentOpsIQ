# Legal And IP Operating Boundaries

This file is not legal advice. It is a project operating checklist.

## Immigration And Work Boundary

Before incorporation, revenue, customer contracts, or active commercial operation, confirm allowed activity with an immigration attorney.

Clarify:

- Learning and personal prototyping
- Open-source contribution
- Passive ownership
- Incorporation
- Revenue
- Customer support
- Contracting
- Fundraising

## IP Boundary

- Use original code only.
- Use public information only.
- Do not copy proprietary startup code, UI, documentation, screenshots, workflows, or private implementation details.
- Keep all design choices independently authored.
- Track sources for public inspiration.

## Enterprise Trust Boundary

AgentOpsIQ should be positioned around:

- Reliability
- Safety
- Governance
- Auditability
- U.S. enterprise productivity
- Cloud and AI infrastructure value
